# Cross-Platform Music Engagement Dashboard

## Overview
This project explores how engagement on social media platforms such as **TikTok** and **YouTube** influences **Spotify streaming performance**.  
An interactive Tableau dashboard was developed to analyze cross-platform trends, rankings, and correlations between user engagement and music popularity.

The goal of this project is to demonstrate data cleaning, visualization, and analytical skills using real-world style datasets.

---

## Tools & Technologies
- **Tableau** – Interactive dashboard design and visualization
- **Excel** – Data cleaning, preprocessing, and integration
- **CSV datasets** – Multi-platform engagement data

---

## Dataset Description
The dataset includes song-level metrics from multiple platforms, such as:
- Spotify streams and rankings
- TikTok engagement metrics
- YouTube views and popularity indicators

Data from  Kaggle was cleaned, standardized, and merged to enable accurate comparison and analysis.

---

## Dashboard Features
- **Interactive filters** to explore individual songs and platforms
- **KPI cards** displaying key engagement and ranking metrics
- **Scatter plots** analyzing correlation between social media engagement and streaming performance
- **Platform dominance analysis** highlighting which platform contributes most to song popularity
- **Consistent color encoding** inspired by platform branding for clarity

---

## Key Insights
- Songs with higher engagement on TikTok often show increased Spotify streaming activity
- YouTube views tend to correlate with long-term streaming consistency
- Cross-platform dominance varies by song, indicating different promotional strengths across platforms
